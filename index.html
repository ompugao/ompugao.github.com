Python and Ruby, English and Japanese

Nov 29th, 2012

まぁよくあるネタなんでしょうが。

例えば

python:
    sorted([4,3,2,1])
ruby:
    [4,3,2,1].sort

を

python:
    I'll *sort* the array, [4,3,2,1].
ruby:
    配列[4,3,2,1]をsortする

とか

python:
    len([3,2,1,4])
ruby:
    [3,2,1,4].length

を

python:
    the *length* of the array,[3,2,1,4]
ruby:
    配列[3,2,1,4]の長さ

とか

python:
    ' '.join(["hoge","fuga","piyo"])
ruby:
    ["hoge","fuga","piyo"].join(' ')

を

python:
    the charactor ' ' joins the array,["hoge","fuga","piyo"]
ruby:
    配列["hoge","fuga","piyo"]を文字' 'でつなぐ

みたいにpythonは英語,rubyは日本語で読めばいいんだと思った。

Tmux Setting

Oct 10th, 2012

type your password only once

Add the following function to .zshrc.

function savepasswd(){
    local password
    echo -n "password:"
    read -s password ; echo
    tmux bind-key y send-keys "$password"
}

Then,call the function and save your password.

And then, type: C-<your tmux prefix> y.
